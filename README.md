# manhwaread-downloader

A simple python tool to download manhwas from https://manhwaread.com

## Installation
Install python, download the script, install the dependencies:
`pip install curl-cffi`

## Usage:
`python downloader.py <url1> <url2> <list.txt>`  
You can provide one or multiple urls or one or multplies files.


You can then convert it to a PDF using [images-to-pdf](https://github.com/mostlikelyadev/images-to-pdf)
